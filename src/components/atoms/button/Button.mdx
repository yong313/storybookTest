## Prop
Button Components에서 사용되는 Props 입니다.

- <span style={{...point}}>variant</span> props로 정해진 style의 button만 사용


- <span style={{...point}}>size</span> props로 정해진 button size만 사용


- <span style={{...point}}>custom</span> 필요 시 <span style={{...point_02}}>fontSize</span> <span style={{...point_02}}>fonstColor</span> <span style={{...point_02}}>bgColor</span> 를 사용해 폰트의 크기, 색상, 버튼색상만 custom 가능


- <span style={{...point}}>disabled</span> boolean으로 조절


- <span style={{...point}}>onClick</span> button click event 적용
